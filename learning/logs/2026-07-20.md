### 11. if/else com &&

```javascript
const idade = 18;
const temCarteira = true;

if(idade >= 18 && temCarteira === true){
    console.log("Ok, já pode dirigir")
} else {
    console.log("Não pode dirigir ainda")
}
```

### 12. Simplificação da condição booleana

```javascript
const idade = 18;
const temCarteira = true;

if(idade >= 18 && temCarteira){
    console.log("Ok, já pode dirigir")
} else {
    console.log("Não pode dirigir ainda")
}
```

> Como o && é um operador lógico de AND, ele verificaria se ambas atendem as condições. Se não, ele iria direto para o else.
